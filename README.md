## FOR RUN SERVER INSTALL JAVA 8
## How to run server?
  - Run with this command and stop after first run

```sh
$ java -Xmx2G -Xms512M -jar server.jar
```
  - Edit server.properties (which is created after first run)
  - Then Run again server same code
